# MiniMart

A minimal Kotlin/Spring Boot microservices order-placement platform — `identity-service`, `catalog-service`, `order-service`, `notification-service`, each with its own database and no shared schema. This README indexes the project's design and decision documents under `Archive/`.

## Documentation governance

- **One-time setup:** run `./scripts/setup-hooks.sh` to turn on the doc-audit pre-commit gate for your clone (opt-in, not automatic — see [Governance](Archive/GOVERNANCE.md) §5 for what it does).
- **[Governance](Archive/GOVERNANCE.md)** — how `Archive/` is organized, the required doc header, the ChangeRequest/ChangeLog template, and the automated review layers (structural audit script, on-demand `/doc-review`, scheduled drift review).
- **[Index](Archive/INDEX.md)** / **[Rule-ID cross-reference](Archive/RULE-INDEX.md)** — generated by `scripts/doc-audit.sh --index`; regenerate after adding or editing an Archive doc rather than hand-editing these two files.
